<h1>Nodeheim - Network Discovery for Splunk</h1>

<p>Nodeheim is a standardized network discovery and analysis app that implements Splunk's standard action types for network operations. It provides interoperable network scanning, topology analysis, and comparison capabilities while maintaining compatibility with both Free and Enterprise editions.</p>

<h2>Key Features</h2>
<ul>
    <li>Standardized network discovery actions</li>
    <li>Interoperable with other network tools</li>
    <li>Resource-optimized for Free Edition</li>
    <li>Built-in visualization support</li>
</ul>

<h2>Standard Actions</h2>
<ul>
    <li><strong>discover_hosts</strong>: Network host discovery</li>
    <li><strong>investigate_network</strong>: Network topology analysis</li>
    <li><strong>compare_networks</strong>: Network change detection</li>
</ul>
